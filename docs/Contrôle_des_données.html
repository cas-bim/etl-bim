Contrôle des données

Objectif

Dans cet exercice, nous allons contrôler des données REVIT.

Nous devons valider que les portes possèdent les caractéristiques suivantes :

  • Si portes extérieures, leurs types doivent être :
    • 0915 x 2134mm
    • M_Curtain Wall Sgl Glass
  • Si portes intérieures, leurs types doivent être : ?? à voir
    • 0915 x 2134mm

Conclusion

Quantifier la qualité des données est un élément crucial pour garantir le succès d’un projet et s’assurer de son utilisation. FME permet de développer rapidement des checkers de données, tant attributaires que géométriques permettant de déterminer l’adéquation des données par rapport à un prérequis (normes, modèle de données, règles métier).

De plus, FME permet une publication en ligne des checkers pour une utilisation publique et (potentiellement) gratuite. Quelques exemples de checkers FME :

  • Checker SIRE, Canton de Vaud : https://www.geopol.ch/#/fr/workspaces/checker-sire
  • Checker Mutation, Canton de Genève : https://www.geopol.ch/#/fr/workspaces/checker-mutation-geneve
