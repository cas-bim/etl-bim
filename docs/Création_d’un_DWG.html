Création d’un DWG

Objectif

Depuis un modèle numérique de terrain (MNT), créer des courbes de niveau, puis les enregistrer dans un fichier DWG.

Ressource

Workspace :

  •  ex_creation_dwg \ ex_creation_dwg_start.fmw, nous débutons l’exercice avec un workpace vide.

Données :

  •  ex_creation_dwg \ressources\ MN_Terrain_GRID.asc – modèle numérique de terrain.

Exercice

Démarrer FME Workbench

Cliquer sur FME Desktop > FME Workbench dans le menu démarrer.

Tâches avancées

De façon autonome, essayer de :

  • Changer la couleur des courbes de niveau
  • Placer les labels dans un autre calque
  • De ne pas transmettre de polyligne 3D dans le DWG (uniquement en 2D)
