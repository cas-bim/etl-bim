Extraction de données depuis un DWG

Objectif

Récupérer des géométries d’un fichier DWG et créer un PDF.

Ressource

Worskpace :

  • \ ex_extraction_dwg\ ex_extraction_dwg_start.fmw, nous débutons avec un workspace vide.

Données :

  • \ ex_extraction_dwg\ ressources\Roads.dwg – fichier dwg contenant le tracé de routes dans différents calques.  

Exercice

Démarrer FME Workbench.

Cliquer sur FME Desktop > FME Workbench dans le menu démarrer.

Sélectionner Generate.

Sur l’interface principale, sélectionner « generate ».


Ajouter le fichier input (reader).

Naviguer dans la liste des formats disponibles, sélectionner le format Autodesk AutoCAD DWG/DXF.

Sélectionner le fichier input AutoCAD Roads.dwg.

Définir le fichier output (writer)

Sélectionner Adobe 3D PDF pour le format du Writer, puis entrer le chemin d’accès du futur fichier à créer.

En validant, la fenêtre suivante doit apparaître :

Exécuter le script.

En cliquant sur .

Ouvrir le fichier PDF créé.

Attention, il faut valider les paramètres de sécurité.

Changer la couleur du calque « Private »

Ajouter un « FeatureColorSetter » et tapant ce mot clé dans la fenêtre principale.

Le nouveau transformer apparaît sur le canevas. Changer la couleur en double cliquant dessus.

En déplaçant le transformer, vous pouvez l’inclure entre le reader et le writer du calque « Private ». Ceci peut aussi s’effectuer en tirant les liaisons vers le transformer.

Le workspace doit ressembler à cela :

Exécuter le script.

En cliquant sur .

Ouvrir le fichier PDF créé.

Contrôler que la couleur a bien été changée.
