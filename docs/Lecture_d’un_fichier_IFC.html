Objectif

Ressource

Workspace :

  • exo\ ex_lecture_ifc\ ex_lecture_ifc_start.fmw, nous débutons l’exercice avec un workspace vide

Données :

  • exo\ ex_lecture_ifc\ ressources\Duplex_A.ifc – Fichier IFC, modélisation d’une maison
